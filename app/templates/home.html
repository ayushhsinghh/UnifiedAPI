All endpoints are rooted under /api. Example requests below:

Create Transcription Job

POST /api/jobs

Upload a video file and start transcription

Get Job Status

GET /api/jobs/{job_id}

Check the status and progress of a transcription job

Download Subtitles

GET /api/jobs/{job_id}/subtitles

Download the generated subtitle file (SRT format)

List All Jobs

GET /api/jobs

Retrieve a list of all transcription jobs

Delete Job

DELETE /api/jobs/{job_id}

Delete a transcription job and its files

üìù Core Endpoints

Create Game

POST /api/game/create

Create a new game session with a category

Join Game

POST /api/game/{session_id}/join

Join an existing game with session code

Start Game

POST /api/game/{session_id}/start

Start the game and assign imposter (minimum 2 players required)

Get Game State

GET /api/game/{session_id}?player_id={player_id}

Get current game state and player information (shows different topic based on role)

Submit Vote

POST /api/game/{session_id}/vote?player_id={player_id}

Vote for suspected imposter during voting phase

Transition to Voting

POST /api/game/{session_id}/transition-voting

End discussion phase and start voting phase

End Voting & Get Results

POST /api/game/{session_id}/end-voting

End voting and determine game results

List Available Games

GET /api/games/available

Get list of all games waiting for players

Start New Round

POST /api/game/{session_id}/new-round

Start a fresh round for the same players (creator only)

Get Game Result

GET /api/game/{session_id}/result

Retrieve the reveal/result after voting (shows voted out player and winners)

Player Heartbeat

POST /api/game/{session_id}/heartbeat

Keep a player marked active in a session (body: {"player_id":"..."})

Cleanup Inactive Players

POST /api/games/cleanup-inactive

Trigger a cleanup pass to remove inactive players from waiting games

Cleanup Old Games

POST /api/games/cleanup

Delete old/waiting game sessions (server-side maintenance)

Delete Game

DELETE /api/game/{session_id}

Delete a game session and all its players/data
